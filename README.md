# Narayana Industries Company Website

This is a basic company website for Narayana Industries, featuring a company profile and Google integration.

## Features

- Homepage with company introduction
- About section with company profile
- Services section
- Contact section with Google Maps integration
- Responsive design
- Smooth scrolling navigation
- Google Fonts integration

## Google Integration

- **Google Fonts**: Used for typography (Roboto font)
- **Google Maps**: Embedded map showing company location in Hyderabad, India
- **Google Analytics**: Placeholder for tracking (requires actual tracking ID)

## Usage

1. Open `index.html` in a web browser to view the website.
2. Customize the content in `index.html` as needed.
3. Modify styles in `styles.css`.
4. Add interactivity in `script.js`.

## Customization

- Replace placeholder content with actual company information
- Update the Google Maps embed URL with the correct location
- Add a real Google Analytics tracking ID in `script.js`
- Add images or logos as needed

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Google Maps Embed API
